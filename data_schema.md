# Data Schema

This study uses a patient-level baseline dataset (N = 198) with the following variables:

## Demographics

* gender (binary)

## Anthropometrics

* height_cm (continuous)
* weight_kg (continuous)
* bmi (continuous)

## Clinical Conditions

* has_fatty_liver (binary)

## Laboratory Measures

* lab_hba1c (continuous)
* lab_alt (continuous)
* lab_tg (continuous)

## Outcomes

* visceral_fat_area (continuous)
* bapwv_mean (continuous)

## Symptom Measures

* tcm_fatigue (ordinal)

All data were de-identified prior to analysis.
